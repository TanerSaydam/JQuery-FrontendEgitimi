JQuery

JQuery Nedir?

Dünyada en çok kullanılan Javascript kütüphanesidir. Amacı Javascript dilini internet üzerinde kolayca kullanabilmeyi sağlamak.

JQuery'nin sağladığı en büyük avantjalardan biri, tutarlı bir şekilde bütün tarayıcılarda çalışasıdır.

Ne işe yarar?

JQuery, hızlı, küçük, çapraz platformlu ve zengin özelliklere sahip bir Javascript kütüphanesidir.

Neden kullanmalıyız?

Birçok Javascript kütüphanesi olmasına rağmen JQuery bunların arasından en çok tercih edilen kütüphane olmasının arkasındaki nedenleri aşağıdaki şekilde listeyebiliriz.

  1. Küçük boyutludur. Jquery jütüphanesinin küçük boyutlu olması uygulamarı yavaşlatmayacağından dolayı tercih edilir.
  2. Açık kaynak kodludur. Açık kaynak kodlu olmasından dolayı binlerce geliştirici JQuery ile eklenti geliştirebilmektedir.
  3. Cross-Browser Desteği. JQuery farklı tarayıcılardan başarılı şekilde çalışmaktadır. Bu nedenle çokça tercih edilir.
  4. Kolay öğrenilebilmesi. "Az kodla fazlasını başar" felsefesini bensimsediğinden dolayı çok kullanışlıdır.

Sağladığı kolaylıklar Neledir?

  1. HTML/DOM manipülasyonu yapabilir
  2. CSS manipülasyonu yapabilir
  3. Animasyonlar ve efektler yapabilir
  4. AJAX İşlemleri yapabilir
  5. Event'leri oluşturmak ve yönetmek
